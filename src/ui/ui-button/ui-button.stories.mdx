# Button

En este apartado podrás ver todas las variantes del componente `Button`, el cual está diseñado para 
recibir diferentes propiedades que permitiran ver reflejadas sus diferentes variantes, a continuación podrás
ver cada una de ellas.

- Botón `default`, `default disabled`, `default icon`. Es la variación más simple del componente `Button`, en su estado `default`, `disabled`
y con un icono con la propiedad `icon` del tipo `ReactNode`:

<Canvas>
    <Story
        name="Button Default"
        args={{
            ...defaultArgs,
            children: 'Default',
            styles: {
                width: '150px'
            }
        }}>
        {Template.bind({})}
    </Story>
    <Story
        name="Button Disabled"
        args={{
            ...defaultArgs,
            children: 'Disabled',
            disabled: true,
            styles: {
                width: '150px'
            }
        }}>
        {Template.bind({})}
   </Story>
   <Story
        name="Button Icon"
        args={{
            ...defaultArgs,
            children: 'Icon',
            icon: <span className="icon-check" />,
            styles: {
                width: '150px'
            }
        }}>
        {Template.bind({})}
   </Story>
</Canvas>

# API

| Property     | **Description**                                                                  | Is required | Type                                            | Default |
|--------------|----------------------------------------------------------------------------------|-------------|-------------------------------------------------|---------|
| `appereance` | **Opción para seleccionar el tema del botón**                                    | `false`     | `rounded`, `outline`, `link`, `expert`, `beauty` | -       |
| `className`  | **Opción para agregar una clase css que nos permita customizar el botón**        | `false`     | `string`                                        | -       |
| `children`   | **Permite establecer contenido incluído dentro del componente**                  | `true`      | `string` ó `ReactNode`                          | -       |
| `disabled`   | **Permite habilitar o deshabilitar el funcionamiento del botón**                 | `false`     | `boolean`                                       | `true`  |
| `onClick`    | **Nos permite ejecutar una función cuando damos click al botón**                 | `true`      | `(e: MouseEvent<any>) => void`                  | -       |
| `isLoading`  | **Establece el estado de carga del botón**                                       | `false`     | `boolean`                                       | `false` |
| `icon`       | **Permite agregar un icono al botón**                                            | `false`     | `ReactNode`                                     | -       |
| `styles`     | **Permite customizar el botón en su ancho, alto, color, fondo, radio del borde** | `false`     | `StylesType`                                    | -       |



#### Styles (StylesType):

| Property       | Is required | Type     | Default |
|----------------|-------------|----------|---------|
| `background`   | `false`     | `string` | -       |
| `color`        | `false`     | `string` | -       |
| `borderRadius` | `false`     | `string` | -       |
| `width`        | `false`     | `string` | -       |
| `height`       | `false`     | `string` | -       |
| `borderColor`  | `false`     | `string` | -       |
