# Input

En este apartado podrás ver todas las variantes del componente `Input`, el cual está diseñado para 
recibir diferentes propiedades que permitiran ver reflejadas sus diferentes variantes, a continuación podrás
ver cada una de ellas.

- Input `default` esta es la variación más simple de este componente donde solo recibe como propiedades 
`label` del tipo `string` y una función en su propiedad `onChange` que nos permitirá ver reflejados
los cambios de entrada del componente:

<Canvas>
    <Story
        name="Input Default"
        args={{
            label: 'Input default',
            onChange: () => {},
        }}>
        {Template.bind({})}
    </Story>
</Canvas>

- Input con un valor en su propiedad `value`:

# API

| Property       | **Description**                                                                | Is required | Type                                        | Default   |
|----------------|--------------------------------------------------------------------------------|-------------|---------------------------------------------|-----------|
| `className`    | **Opción para agregar una clase css que nos permita customizar el componente** | `false`     | `string`                                    | -         |
| `hint`         | **Permite agregar una insinuación debajo del componente**                      | `false`     | `string`                                    | -         |
| `label`        | **Representa el label del input**                                              | `false`     | `string`                                    | -         |
| `name`         | **Representa el nombre del input**                                             | `false`     | `string`                                    | -         |
| `value`        | **Representa el valor del input**                                              | `false`     | `string`                                    | -         |
| `style`        | **Representa el estilo del input**                                             | `false`     | `outline`, `filled`, `view`                 | `outline` |
| `type`         | **Tipo de input**                                                              | `false`     | `text`, `number`, `password`                | `text`    |
| `maxLength`    | **Establece la longitud máxima permitida en el input**                         | `false`     | `number`                                    | -         |
| `minLength`    | **Establece la longitud mínima permitida en el input**                         | `false`     | `number`                                    | -         |
| `disabled`     | **Habilita o deshabilita el input**                                            | `false`     | `boolean`                                   | -         |
| `prefix`       | **Nos permite agregar un prefijo al input**                                    | `false`     | `string`                                    | -         |
| `placeholder`  | **Representa el placeholder del input**                                        | `false`     | `string`                                    | -         |
| `hasError`     | **Habilita el modo error en el input**                                         | `false`     | `boolean`                                   | -         |
| `errorMessage` | **Nos permite agregar un mensaje en el modo error del input**                  | `false`     | `string`                                    | -         |
| `success`      | **Habilita el modo success en el input**                                       | `false`     | `string`                                    | -         |
| `onFocus`      | **Devolución de llamada en modo focus del input**                              | `false`     | `(e:ChangeEvent<HTMLInputElement>) => void` | -         |
| `onBlur`       | **Devolución de llamada cuando el input pierde su estado focus**               | `false`     | `(e:ChangeEvent<HTMLInputElement>) => void` | -         |
| `onChange`     | **Devolución de llamada cuando el estado del input ha cambiado**               | `true`      | `(e:ChangeEvent<HTMLInputElement>) => void` | -         |
